My Course Experience

Personal Reflection

This course is the perfect introductory course to the world of computer science and computer programming. COMP1511 has been my favourite university course throughout my time at university, from the organisation of the lectures and tutorials, to the structure of the course content, to the fun and interesting problems presented in assessments.

Topics covered in course

Fundamental programming and computer science concepts

  • Computer Memory: Random Access Memory (RAM), Bits and Bytes, and the use of Variables
  • Operating Systems: Linux as Unix-based, open source operating system
  • Interaction between the Terminal and Shell
  • Shell Commands: ls, mkdir, cd, pwd, clear

The C programming language and use of a C compiler

  • Basic Syntax: writing comments \\, import statements, newline indicator ;, operators
  • Functions: the main function and other inbuilt functions such as printf(), scanf()
  • Assignment and Variables: creating constants using #define
  • Conditionals: if() else()
  • Compilation: working in the linux terminal using gcc
  • Programming Style: use of intrinsic and internal documentation, whitespace, and principles such as KISS and DRY

Datatypes and Structures

  • Custom Datatypes: Structs, Enums
  • Arrays: declaring arrays, working with 2D arrays, and arrays of structs
  • Strings: using string functions, reading strings with fgets(), and outputting strings with fputs()

Dynamic Memory Management

  • Dynamic Memory: HEAP, Stack, using malloc()
  • Pointers: references and dereferencing variables
  • Linked Lists: working and applying linked lists

Multi File Projects

A brief introduction into different file types such as .h and .c files

My Course Experience

Personal Reflection

COMP2521 is a course that begins to scratch the surface of more advanced computer science concepts. Oftentimes introducing theoretical concepts and then demonstrating their application to real-world scenarios, which highlight the fascinating level of complexity implemented in current modern software systems. The course ties in mathematics concepts in the form of Graph Theory and Big-O Notation, which provides a unique algebraic perspective on computer science topics. Topics such as performance costs analysis and abstraction (ADTs) are cleverly weaved together and interlinked to other topics in the course to allow for a deeper understanding of their application and relevance in other areas of computer science. analysis and abstract data types.

Topics covered in course

Compilation and Makefiles

  • Operating the Gnu C Compiler (GCC): Understanding compilation stages including Pre-processing, Compiling and Linking
  • Utilising Makefiles: specifying the target, source, and commands. Using flags such as -Wall, -Werror, -O, and creating variables using the $ operator

Performance Analysis: Big-O

  • Writing algorithms and pseudocode
  • Measuring literal program run time using the time command in the terminal.
  • Big-O Notation: asymptotic analysis of algorithms
  • Complexity Classes: Polynomial Time (P, Tractable) and Non-polynomial Time (NP, Intractable) and Non-computable problems.
  • Coarse-grained and Fine-grained performance analysis
  • Identifying hot spots and execution profiles
  • Using the gprof Profiler: creating Flat Profiles and Call Graphs

Abstract Data Types

  • Setting up ADTs: creating ADT interfaces in .h files and implementations in .c files
  • Collections - Set ADTs: creating a collection of operations, setting Pre/Post conditions

GDB and Valgrind

  • Using portable debuggers such as GDB: setting and executing through break points in code
  • Detexting memory leaks using Valgrind

Trees

  • Binary Search Trees (BST) - ADT implementation
    • Insertion of a tree node
    • Tree Traversal / Search: Preorder, In-order, Postorder, and Level Order traversal
    • Joining two BSTs
    • Removing a node from a tree
  • Balanced Binary Search Trees: Rotations, Partitions, and Root Insertion
  • AVL Trees
  • 2-3-4 Trees

Graph Theory

  • Graph Representation: Array of Edges, Adjacency Matrix, Adjacency List
  • Graph Traversal: Bread-first Search and Depth-first Search
  • Graph Algorithms:
    • Cycle Checking: identifying connected components
    • Hamiltonian Paths and Circuits
    • Euler Paths and Circuits
  • Directed Graphs (Digraphs) and Weighted Graphs
    • Vertex Degree
    • Directed Paths
    • Vertex Reachability and Strong connectivity
    • Directed Acyclic Graphs
  • Minimum Spanning Trees
    • Kruskal's Algorithm
    • Prim's Algorithm
  • Shortest Path Problems
    • Problem Variations: Source-Target, Single-Source, and All-Pairs
    • Edge Relaxation process
    • Dijkstra's Algorithm

Sorts

  • Sorting Algorithm Properties: Stability and Adaptability
  • O(N2) Sorting Algorithms
    • Selection Sort
    • Bubble Sort
    • Insertion Sort
    • Shellsort
  • O(N log2(N)) Sorting Algorithms
    • Mergesort
    • Quicksort
    • Heapsort

Heaps and Priority Queues

  • Heap Insertion and Deletion
  • Priority Queue: application of heap insertion and deletion to a queue structure

Hashing

  • Ordered and Associative Containers
  • ADT Implementation and Hash Operations: Insertion, Hash Generation, Hash Deletion
  • Collision Resolution: Seperate Chaining, Linear Probing, Double Hashing

Tries

  • Possible Implementations of Trie Searches, Insertion, and Deletion
  • Compressed Tries

Assessments

  • URL Network Graph Representation and Page Rank Calculation: Program that reads data from a collection of page URLs and builds a graph structure using an adjacency list, then calculates the 'Page Rank' of a URL in the collection based on certain criteria such as number of outgoing and incoming links.
